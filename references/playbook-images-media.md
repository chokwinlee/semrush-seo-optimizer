# Playbook: Images and Media

## Target Issues

`13,14,110`

## Root Causes

- dead image references
- missing alt attributes
- externally hosted media instability

## Allowed Auto Actions

- fix local image paths
- remove broken image references when replacement is unavailable
- add meaningful alt text where safe and unambiguous

## Required Validation

- no broken image references on edited pages/templates
- alt coverage improved for informative images
